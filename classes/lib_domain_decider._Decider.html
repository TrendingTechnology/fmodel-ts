Class _Decider<C, Si, So, Ei, Eo>

_Decider is a datatype that represents the main decision-making algorithm. It has five generic parameters C, Si, So, Ei, Eo , representing the type of the values that _Decider may contain or use. _Decider can be specialized for any type C or Si or So or Ei or Eo because these types does not affect its behavior. _Decider behaves the same for C=Int or C=YourCustomType, for example.

_Decider is a pure domain component.

property

decide - A function/lambda that takes command of type C and input state of type Si as parameters, and returns/emits the list of output events Eo[]>

property

evolve - A function/lambda that takes input state of type Si and input event of type Ei as parameters, and returns the output/new state So

property

initialState - A starting point / An initial state of type So
